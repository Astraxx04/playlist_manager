Project Details

  1. Create new song: This endpoint is used to add a new song entry in the songs table.
  2. List available songs: This endpoint is used to list all the available songs in the app.
  3. Create new playlist: This endpoint is used to add a new playlist entry in the playlists table.
  4. List available playlists: This endpoint is used to list all the available playlists in the app.
  5. Edit playlist metadata: This endpoint is used to change the name of an existing playlist.
  6. Delete playlist: This endpoint is used to delete an existing playlist.
  7. List playlist songs: This endpoint is used to list all the songs associated with a playlist.
  8. Move playlist song: This endpoint is used to move a song up and down in a playlist i.e., reposition it.
  9. Remove playlist song: This endpoint is used to remove a song from a playlist.
